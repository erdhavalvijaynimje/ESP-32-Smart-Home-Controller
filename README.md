# ESP32 Smart Room Controller

An IoT based project using ESP32 WebServer.

## Features

- LED ON/OFF control
- Multiple LED control
- PWM brightness control
- WiFi web control

## Hardware

- ESP32 DevKit V1
- LEDs
- Resistors
- Jumper wires

## Learning

- WiFi.softAP()
- WebServer library
- HTML forms
- PWM
